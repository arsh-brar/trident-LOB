# Experiments

Reproducible experiment notes, configs, metrics summaries, and generated reports. Do not store API keys, broker secrets, or paid raw data here.

Local experiment tracking artifacts belong under ignored runtime directories
such as `experiments/mlruns` and append-only audit logs. Shareable summaries may
be exported only when they are secret-free, paid-payload-free, and free of
private account information.

Source: `research/14-reproducibility-and-experiment-tracking/DECISION.md`.
