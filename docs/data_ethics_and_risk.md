# Data Ethics And Risk

TRIDENT-LOB is a research, backtesting, and paper-trading project first. Phase 0 and Phase 1 must not place live trades.

## Policies

- Do not commit API keys, broker credentials, paid data, or private account data.
- Do not use data in a way that violates provider licenses.
- Do not use future data in any feature, label, news score, or validation split.
- Do not claim profitability without out-of-sample, transaction-cost-adjusted evidence.
- Treat delayed data, exchange rules, broker restrictions, and corporate actions as first-class constraints.
